# Trackli

React + Vite NFP/Zykluskalender-App (German), Daten lokal in IndexedDB, kein Backend/Login. Deployed via GitHub Actions nach `https://tinaschickle.github.io/trackli/`.

## Workflow

- Änderungen nach jedem abgeschlossenen Task direkt committen und nach `origin/main` pushen, ohne vorher nachzufragen.
